# NGTech Logo

## Instructions

Placez votre fichier logo officiel dans ce dossier avec le nom exact :

```
logo.png
```

## Emplacement

Le logo sera utilisé dans les composants suivants :
- Navigation (barre de navigation)
- HeroSection (section d'accueil)
- Footer (pied de page)

## Tailles utilisées

- Navigation : 48x48px (w-12 h-12)
- HeroSection : 96x96px (w-24 h-24)
- Footer : 64x64px (w-16 h-16)

## Format recommandé

- Format : PNG avec transparence
- Résolution : Minimum 200x200px pour une bonne qualité
- Fond : Transparent pour s'intégrer parfaitement

Le logo sera automatiquement redimensionné avec `object-contain` pour maintenir les proportions.
